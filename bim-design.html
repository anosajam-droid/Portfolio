01 —

Quantity Take-Off, Cost & Embodied Carbon

Revit Schedules → 5D Analysis

Three building materials were assessed across the complete QTO pipeline. Quantities were extracted directly from the Revit 2024 model as material schedules — no manual counting. Each schedule was exported as CSV, cleaned in Excel, and analysed for cost and embodied carbon using density data and EE/EC intensity values from industry datasets.

The Glass schedule (shown opposite) demonstrates the workflow in action: 25 window and door elements, each with family type, count, material name, unit weight, area, and volume — all extracted automatically from the model. This same process was applied to Concrete Masonry (walls) and Wood/Timber (roof structure).

The pipeline demonstrates a complete Revit → CSV → Excel → 5D BIM → sustainability data chain — the same workflow used on live projects to generate cost plans and carbon reports from federated models.

02 —

4D Construction Simulation

Fig 2. Navisworks TimeLiner — full 11-task programme. Foundation (30/03) through Railing Fabrication (01/10/2024). All tasks linked to model elements.

Navisworks TimeLiner — 185-Day Programme

A 4D simulation was developed by linking Revit model elements to an 11-task construction programme in Navisworks TimeLiner. The programme spans 185 days — from Foundation (30 March 2024) to Railing Fabrication (1 October 2024) — with each task linked to the corresponding building elements in the 3D model.

As the simulation runs, each active task highlights green in the 3D view, making the construction sequence visually legible. The programme reflects realistic construction logic: structure first (foundations → slab → columns → walls), then envelope (roof → doors → windows), then finishes.

The Gantt chart (shown opposite) displays all 11 tasks with planned start and end dates, linked directly to the model's selection tree which contains 24 element categories — Walls (44), Windows (24), Doors (13), Columns (4), Structural Foundations, and more.

Simulation in Progress — First Floor Walls

The screenshot opposite shows the simulation at Day 150 (27 August 2024) — the First Floor Walls task at 25.99% completion. The green highlight shows the active task elements being placed in the model in real time, while the completed lower structure (ground floor slab, columns, walls, first floor slab) is shown in white.

This visual communication of construction sequence is one of the core benefits of 4D simulation — it allows project teams, clients, and contractors to understand exactly what is being built and when, reducing RFIs and sequencing conflicts before they reach site.

The simulation ends at Day 185 (1 October 2024) with all elements complete — windows (35.6%), doors (52%), and railing fabrication all installed, producing a fully rendered model of the completed dwelling.

03 —

Clash Detection — MEP vs Structural

50 Clashes — MEP vs Structural IFC Federation

Two IFC models were federated in Navisworks Manage: MEP_HVAC.ifc and Structural_Model.ifc. Both models were set to identical project origins before running the test — critical for accurate cross-discipline coordination without false positives from misaligned coordinate systems.

A Clearance clash test was run at 0.001m (1mm) tolerance — the standard threshold for MEP/structural coordination. The test identified 50 total clashes, all clearance type, at distances ranging from −0.003m to 0.000m. All even-numbered clashes (Clash2, Clash4 ... Clash50) were grouped as SameClashType, status set to Reviewed, and assigned to Jambaldinni-19291526.

A full HTML clash report was generated and exported (19291526_Jambaldinni), recording all 50 clashes with status, level, grid intersection, found date, description, assigned party, and distance values.
