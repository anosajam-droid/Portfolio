Clash Detection — MEP vs Structural

Power BI Data Pipeline

A full Revit-to-Power BI data pipeline was developed to demonstrate how BIM model data can be transformed into interactive project intelligence — eliminating manual extraction and enabling real-time cost, carbon, and quantity monitoring.

Asset generation. Schedules created per material: Glass, Wood, Concrete Masonry

CSV Export

◉
Excel Clean-up
Data cleaned and structured. EE and CO₂ calculations applied per element
◐
Power BI Import
Excel workbook imported. Data model relationships defined across material types
▣
Visualisation
Interactive dashboards: cost, QTO and carbon metrics per building element
